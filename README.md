# Projeto " Meu jogo 2D em java "

O projeto **MyApp2D** é um jogo criado em java puro sem uso de bibliotecas externas.

### Conceitos que vão ser aprendidos:

- Aprendizado de gameLoop
- Ciclo de atualização e desenho
- Entrada pelo teclado (W,A,S,D)
